Explore Lists and Loops in Python

In Python, a list is a versatile data structure used to store collections of items. Lists are mutable, meaning you can modify, add, or remove elements from them after creation. Lists are denoted by square brackets [] and elements inside a list are separated by commas.

Common List Operations:

  • Accessing Elements: You can access individual elements in a list using their index. Indexing starts from 0 for the first element, 1 for the second element, and so on. Negative indexing allows you to access elements from the end of the list, starting with -1 for the last element.
  • Slicing: You can create a sublist (slice) by specifying a range of indices. Slicing allows you to extract a portion of the list.
  • Appending and Extending: You can add elements to the end of a list using the append() method. To add multiple elements from another iterable (e.g., another list), you can use the extend() method.
  • Inserting: You can insert elements at a specific index using the insert() method.
  • Removing Elements: You can remove elements from a list using the remove() method or by using the pop() method to remove an element at a specific index.
  • Looping through Lists: Loops, such as for and while, are used to iterate through lists and perform operations on each element.

Examples of Looping Through a List:

Using a While Loop:

        numbers = [1, 2, 3, 4, 5]
        i = 0
        while i < len(numbers):
            print(numbers[i])
            i += 1

Using a For Loop:

        numbers = [1, 2, 3, 4, 5]
        for num in numbers:
            print(num)
